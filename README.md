# Book Tracker

**Автор:** [Ваше Имя и Фамилия]

## Описание
Приложение для управления списком прочитанных книг. Позволяет добавлять новые книги, указывать их автора, жанр и объем, а также фильтровать накопленный список.

## Функционал
- Добавление книг с валидацией данных.
- Фильтрация по жанру и количеству страниц.
- Автоматическое сохранение данных в файл `books.json`.

## Инструкция по запуску
1. Установите Python (версии 3.x).
2. Скачайте файл `main.py`.
3. Запустите программу командой:
   ```bash
   python main.py
